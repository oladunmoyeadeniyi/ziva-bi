Financials sidebar. Categories are derived from the CoA GL group/subgroup
structure. The page and all references to it must be removed. Any navigation
to the old URL is redirected to the Chart of Accounts page.

## CHANGE 2 — `src/app/dashboard/business/settings/expense-categories/page.tsx`

### Replace page content with redirect

Replace the entire file content with a redirect to Chart of Accounts:

```typescript
"use client";

import { useEffect } from "react";
import { useRouter } from "next/navigation";

export default function ExpenseCategoriesRedirect() {
  const router = useRouter();
  useEffect(() => {
    router.replace("/dashboard/business/settings/chart-of-accounts");
  }, [router]);
  return null;
}
```
