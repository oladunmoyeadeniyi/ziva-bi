## CHANGES — `src/app/auth/signup/page.tsx`

### Change 1 — Country dropdown: remove default selection

Find this line:
```typescript
const [companyCountry, setCompanyCountry] = useState("NG");
```

Replace with:
```typescript
const [companyCountry, setCompanyCountry] = useState("");
```

This ensures the country dropdown defaults to blank (`— Select country —`) instead of
pre-selecting Nigeria.

### Change 2 — Country select: add blank option at top

Find the country `<select>` element. It currently maps directly over `COUNTRIES`.
Add a blank first option:

```tsx
<select
  required
  value={companyCountry}
  onChange={(e) => setCompanyCountry(e.target.value)}
  className="w-full rounded-lg border border-gray-300 px-3.5 py-2.5 text-sm text-gray-900 focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent"
>
  <option value="">— Select country —</option>
  {COUNTRIES.map((c) => (
    <option key={c.code} value={c.code}>
      {c.name}
    </option>
  ))}
</select>
```
