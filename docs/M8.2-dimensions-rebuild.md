## CONTEXT

The Dimensions page is being rebuilt from a simple table into a card-based layout
with the following improvements:

1. Each dimension is a collapsible card showing name, code, source badge,
   required badge, value count, and expand/collapse
2. Dimensions have a `value_source` field: manual | org_structure | employee_master
   | customer_category | hybrid | product_master
3. Statistical internal order has a hybrid source with THREE sub-sources:
   employee codes (auto), customer category codes (manual for now), and
   manual codes
4. Each dimension has a toggle (active/inactive) instead of a "Deactivate" button
5. Manual dimensions show Download template + Bulk upload buttons
6. Auto-synced dimensions show a read-only note with source info
7. Master data / values tab shows a dimension selector with appropriate UI per
   source type
8. "Add dimension" opens an inline form with name, code, value_source, and
   required fields
