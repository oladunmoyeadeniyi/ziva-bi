## CONTEXT

Three fixes to the Tax applicability sub-tab:

1. Custom taxes added via the "+ Add" button are not appearing in the UI because
   they are added to `config.tax_items` but the render logic only maps items that
   exist in the base profile groups — custom items (with `custom: true`) are never
   rendered. Fix: render a dedicated "Custom taxes" group at the bottom for all
   items where `custom === true`.

2. When adding a custom tax, the user cannot specify which category it belongs to
   (Indirect taxes, Direct taxes, Employment & payroll, etc.) or create a new
   category. Fix: expand the custom tax input into a small inline form with a
   category selector.

3. The tax list is showing the Generic (XX) profile instead of the Nigeria (NG)
   profile. This means `org.country` is `undefined` or `null` when
   `getTaxGroupsForItems` is called. Fix: use a fallback that reads the stored
   country from the org config correctly.

### Change 1 — Fix country code lookup in tax groups render

In the Tax applicability sub-tab, find this line:
```typescript
const countryCode = org.country ?? "XX";
```

This appears in TWO places in the tax sub-tab (once in the zero-tax haven onChange,
once in the tax groups render). In BOTH places, replace with:

```typescript
const countryCode = (org.country && TAX_PROFILES[org.country])
  ? org.country
  : "XX";
```

This ensures we only use a country code if we actually have a profile for it,
otherwise fall back to generic.

### Change 5 — Render custom taxes at the bottom of the tax groups list

After the closing `})()}` of the tax groups render block (the one that maps
`groups.map(group => ...)`), add a custom taxes group:

```tsx
{/* Custom taxes group */}
{config.tax_items.filter(t => t.custom).length > 0 && (() => {
  // Group custom items by their category
  const customByCategory = config.tax_items
    .filter(t => t.custom)
    .reduce((acc, item) => {
      const cat = item.category ?? "Custom taxes";
      if (!acc[cat]) acc[cat] = [];
      acc[cat].push(item);
      return acc;
    }, {} as Record<string, TaxItem[]>);

  return Object.entries(customByCategory).map(([category, items]) => {
    const isCollapsed = collapsedTaxGroups.has(`custom_${category}`);
    return (
      <div key={`custom_${category}`} className="border border-blue-200 rounded-md overflow-hidden">
        <button
          type="button"
          onClick={() => toggleTaxGroup(`custom_${category}`)}
          className="w-full flex items-center justify-between px-4 py-2.5 bg-blue-50 hover:bg-blue-100 text-left"
        >
          <div className="flex items-center gap-2">
            <span className="text-xs font-semibold text-blue-700 uppercase tracking-wide">{category}</span>
            <span className="text-[10px] bg-blue-100 text-blue-600 px-1.5 py-0.5 rounded">Custom</span>
          </div>
          <div className="flex items-center gap-2">
            <span className="text-xs text-blue-500">{items.filter(i => i.checked).length} of {items.length} selected</span>
            <i className={`ti ti-chevron-${isCollapsed ? "right" : "down"} text-blue-400`} style={{ fontSize: 13 }} />
          </div>
        </button>
        {!isCollapsed && (
          <div className="divide-y divide-blue-50">
            {items.map(item => (
              <div key={item.id} className="flex items-start gap-2.5 px-4 py-2.5">
                <input
                  type="checkbox"
                  className="w-3.5 h-3.5 cursor-pointer accent-blue-600 mt-0.5 flex-shrink-0"
                  checked={item.checked}
                  onChange={e => {
                    setConfig(c => ({
                      ...c,
                      tax_items: c.tax_items.map(t =>
                        t.id === item.id ? { ...t, checked: e.target.checked } : t
                      ),
                    }));
                  }}
                />
                <div className="flex-1 min-w-0">
                  <p className="text-xs font-medium text-gray-900">{item.name}</p>
                  <p className="text-[11px] text-gray-500">{item.desc}</p>
                </div>
                <button
                  type="button"
                  onClick={() => setConfig(c => ({ ...c, tax_items: c.tax_items.filter(t => t.id !== item.id) }))}
                  className="text-red-400 hover:text-red-600 p-0.5 flex-shrink-0"
                  title="Remove custom tax"
                >
                  <i className="ti ti-x" style={{ fontSize: 12 }} />
                </button>
              </div>
            ))}
          </div>
        )}
      </div>
    );
  });
})()}
```
