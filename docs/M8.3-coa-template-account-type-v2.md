### 3. Add cell comment to Account Type header cell

After writing the header row, add a comment to the Account Type header cell:

```python
from openpyxl.comments import Comment

at_col_idx = header_row.index("Account Type*") + 1  # 1-based

at_cell = ws.cell(row=1, column=at_col_idx)
comment = Comment(
    text=(
        "Accepted values:\n"
        "  PL  =  Profit & Loss\n"
        "         (SOCI — Statement of Comprehensive Income)\n\n"
        "  BS  =  Balance Sheet\n"
        "         (SOFP — Statement of Financial Position)\n\n"
        "Both PL/BS and SOCI/SOFP are accepted on upload."
    ),
    author="Ziva BI"
)
comment.width = 280
comment.height = 120
at_cell.comment = comment
```

### 4. Add data validation dropdown for Account Type column

```python
from openpyxl.worksheet.dataval import DataValidation

at_dv = DataValidation(
    type="list",
    formula1='"PL,BS"',
    allow_blank=False,
    showDropDown=False,
    showErrorMessage=True,
    errorTitle="Invalid account type",
    error="Please enter PL (Profit & Loss) or BS (Balance Sheet).",
    showInputMessage=True,
    promptTitle="Account Type",
    prompt="PL = Profit & Loss (SOCI)\nBS = Balance Sheet (SOFP)",
)
at_col_letter = ws.cell(row=1, column=at_col_idx).column_letter
at_dv.sqref = f"{at_col_letter}4:{at_col_letter}5000"
ws.add_data_validation(at_dv)
```
